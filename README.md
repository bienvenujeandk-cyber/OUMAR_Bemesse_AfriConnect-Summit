# AfriConnect Summit 2026

**Projet réalisé par** : Bemesse OUMAR – L1 CS  
**Description** : Site vitrine de la conférence tech panafricaine AfriConnect Summit 2026.  
**Technologies** : HTML5 sémantique, CSS3 (Flexbox, Grid, variables, animations, media queries), JavaScript vanilla (DOM, événements, IntersectionObserver, localStorage, validation).  
**Fonctionnalités JS** :
- Dark mode persistant avec localStorage
- Navbar fixe dynamique (fond après 80px)
- Animations au scroll (fade-in, slide-in, zoom-in)
- Compteurs animés sur les statistiques
- Onglets du programme (3 jours)
- Filtrage dynamique des intervenants par thématique
- Validation de formulaire en temps réel
- Bouton retour en haut
- Année dynamique dans le footer
- Menu hamburger mobile
